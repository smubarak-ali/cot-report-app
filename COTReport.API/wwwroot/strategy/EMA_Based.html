• Strategy Setup:
    • Usage of 3 EMA's (12, 26 and 50), applied to close and shift 0
    • Only Pre-London, London and New York sessions
    • Optional - MACD with only histogram and default settings
  • Long Trade Setup:
    • 15M chart is to mark support and resistance
    • Plot the EMA's and check if all three align properly (i.e. 12 above and 26 above 50)
    • If the alignment is correct and the EMA's are spanned a little bit, drop down to 5M timeframe
    • Here also check the EMA's alignment and if proper look for a pull back to 26 EMA or 50 EMA
    • If MACD is applied then check if the red histogram is smaller and soon the green bar will appear
    • If these conditions met enter the trade with 1:2 RR
  • Short Trade Setup:
    • Inverse / opposite of the long setup
  • Recommendation:
    • Do not risk more than 0.5% per trade
    • Set a number of max trades criteria. For example, if first trade wins then stop trading for the rest of the day. Stop trading when two SL hits in a day, etc.
    • Avoid trading news and exit all the positions before
